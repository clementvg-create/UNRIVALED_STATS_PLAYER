## UNRIVALED_STATS_PLAYER (FRENCH VERSION)

Le dépôt GitHub est organisé de la manière suivante :
 - Dossier /Données --> Données brutes récupérées depuis [Unrivaled Site Web](https://www.unrivaled.basketball/stats)
 - Dossier /Modèle --> Modèle Excel pour exploiter les données brutes
 - Dossier Racine :
	- STATS_2025.xlsx --> Fichier Excel de la saison 2025 des stats (basiques + avancées)
	- STATS_2026.xlsx --> Fichier Excel de la saison 2026 des stats (basiques + avancées)
	- Etc...

# Utilisation

Deux utilisations sont possibles :
 - Avoir Git sur son PC --> Récupération du dépot Git [ici](https://github.com/clementvg-create/UNRIVALED_STATS_PLAYER.git)
 - Télécharger les fichiers Excel (STATS_2025.xlsx ...) en cliquant dessus depuis l'adresse [ici](https://github.com/clementvg-create/UNRIVALED_STATS_PLAYER)

Remarque : Si vous utilisez Git, ne pas faire de modifications sur les fichiers (Commit, Push...), vous pouvez néanmoins Clone/Pull le dépôt à volonté !

## UNRIVALED_STATS_PLAYER (ENGLISH VERSION)

The GitHub repository is organized as follows:
 - Folder /Données --> Raw data retrieved from [Unrivaled Website](https://www.unrivaled.basketball/stats)
 - Floder /Modèle --> Excel template for processing the raw data
 - Root folder:
	- STATS_2025.xlsx --> Excel file for the 2025 season of stats (basic + advanced)
	- STATS_2026.xlsx --> Excel file for the 2026 season of stats (basic + advanced)
	- Etc...

# Usage

Two uses are possible:
 - Having Git on your PC --> Retrieve the Git repository [here](https://github.com/clementvg-create/UNRIVALED_STATS_PLAYER.git)
 - Download the Excel files (STATS_2025.xlsx...) by clicking on them from the address [here](https://github.com/clementvg-create/UNRIVALED_STATS_PLAYER)
 
Note: If you are using Git, do not make any changes to the files (commit, push, etc.). However, you can clone/pull the repository as often as you like!
